# Установщики

В ProjBobcat мы реализовали для разработчиков большую часть инфраструктуры модов, с которой вы можете столкнуться в процессе разработки Minecraft.

## Список поддержки

|      Название проекта       |    Статус поддержки     ||
|:---------------:|:-----------:|:---:|
|   Forge (старая версия)    |     Поддерживается     |  ✅   |
|   Forge (новая версия)    |     Поддерживается     |  ✅  |
|   LiteLoader    |     Поддерживается     |  ✅  |
|     Fabric      |     Поддерживается     |  ✅  |
|    Optifine     |     Поддерживается     |  ✅  |
|      Quilt      |  Поддерживается (экспериментально)   |  ⚠  |
| Модпаки CurseForge  |     Поддерживается     |  ✅  |
