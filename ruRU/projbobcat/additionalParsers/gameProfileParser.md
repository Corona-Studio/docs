# Парсер игровых профилей

Парсер игровых профилей отвечает за запись найденных локальных игр в файл конфигурации официального лаунчера.

## Инициализация парсера

Способ инициализации парсера очень прост:

```c#
var launcherProfileParser
    = new DefaultLauncherProfileParser(rootPath, clientToken)
```

Где **rootPath** — это корневой каталог, в котором находится ядро (т. е. каталог, в котором находится папка .minecraft).
**clientToken** — это случайно сгенерированный GUID.
