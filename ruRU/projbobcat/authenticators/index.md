# Модели аутентификации

В ProjBobcat мы реализовали для разработчиков все сценарии аутентификации, с которыми вы можете столкнуться в процессе разработки Minecraft.

## Список поддержки

|          Название систем           |    Применимый сценарий     |
|:-----------------------:|:-----------:|
|  OfflineAuthenticator   |   Система оффлайн-аутентификации    |
| YggdrasilAuthenticator  | Подходит для старых систем входа в систему |
| MicrosoftAuthenticator  |  Новая система аутентификации Microsoft  |
